PDF export (contract download) needs Unicode fonts so Polish characters (ą, ę, ó, ł, ń, ś, ź, ż, ć) display correctly instead of squares.

Option 1 – Linux: install system fonts (no files here needed):
  sudo apt install fonts-dejavu-core

Option 2 – Place DejaVu TTF files in this folder:
  - DejaVuSans.ttf
  - DejaVuSans-Bold.ttf
  Download from: https://dejavu-fonts.github.io/Download.html
